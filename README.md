This repository contains exploratory work to see if Nengo could be redone on Spinnaker.

The decision was that the effort required is too great for something outside the scope of our current funding.

The work was therefor abandoned as is.

This work is incompletely and left in an unknown probably not easily usable state.

Please see https://github.com/SpiNNakerManchester/nengo_spinnaker for a version done by a PhD student that did work at the time it was done.
